# Analizator-Securitate

Acest proiect a fost scris in Python ,creat să identifice vulnerabilități ,atacuri de tipul brute-force și să evalueze riscuri cibernetice .Poiectul are 3 functionalitati  principale : scanarea porturilor deschise , verificarea log-urilor serverului si monitorizarea starii serverului (memoria) . 

#Scanarea porturilor : Acest modul scanează rețeaua pentru a identifica punctele de acces vulnerabile și serviciile active. Prin funcția scan_ports() verifică starea fiecărui port, detectând dacă este deschis, închis sau filtrat. Comanda identify_service() analizează banner-ul portului pentru a afla ce aplicație rulează, ajutând la depistarea versiunilor învechite. Folosesc check_vulnerabilities() pentru a compara serviciile găsite cu o bază de date de riscuri cunoscute, iar la final funcția display_results() afișează o listă clară cu porturile care reprezintă un pericol imediat.

#Verificare log-uri : Acest script automatizează monitorizarea fișierelor de sistem pentru a detecta activități suspecte în timp real. Scanează logurile după termeni critici (ex: "Failed password", "Invalid user") și mă folosesc de comanda: search_keywords(). După aceea, prin comanda count_incidents() numără frecvența erorilor într-un interval de timp pentru a distinge o simplă greșeală de un atac brute-force. Prin comanda extract_ips() izolează adresele IP care generează alerte, iar în final afișează IP-urile atacatoare pe ecran (generate_report())

#Monitorizare server : Acest modul urmărește resursele serverului pentru a asigura că sistemul funcționează bine și pentru a evita blocajele datorită supraîncărcării. Prin utilizarea funcției monitor_resources(), monitorizează utilizarea procesorului și a memoriei în timp real și identifică orice consum neregulat care ar putea indica un proces malițios. Pentru a detecta fluxuri de date neobișnuite către adrese externe necunoscute, comanda get_active_connections() analizează toate conexiunile de rețea active. Folosesc funcția log_system_status() pentru a salva istoricul performanței, iar funcția alert_admin() alertează dacă parametrii de siguranță sunt depășiți.
